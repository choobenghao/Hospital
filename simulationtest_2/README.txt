How To Run

1. Open FreeMat.

2. Set working directory to the project folder.

3. Run:

HospitalSimulation

4. Main Menu will appear.

5. Select:

1 = Baseline Scenario
2 = Improved Scenario
3 = Compare Scenario
4 = Previous Results
5 = Exit

---------------------------------

Features Implemented

- Priority Queue Scheduling

- Multiple Doctor Support

- Baseline Scenario (3 Doctors)

- Improved Scenario (4 Doctors)

- Peak and Non-Peak Arrival Patterns

- Three Random Number Generators
  - Default rand()
  - LCG
  - Uniform LCG

- Performance Metrics
  - Average Waiting Time
  - Average Queue Length
  - Average System Time
  - Doctor Utilization
  - Probability of Waiting

- Save Simulation Results

- Load Previous Results

- Scenario Comparison
